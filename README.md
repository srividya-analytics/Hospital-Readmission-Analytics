# Hospital Readmission Analytics

## Project Overview

This project analyzes hospital readmission patterns using healthcare data to identify the factors that contribute to patient readmissions. The analysis combines SQL, Python, and Tableau to perform data cleaning, exploratory data analysis, database querying, and interactive dashboard development.

The objective is to generate actionable insights that can help healthcare organizations reduce readmission rates, improve patient outcomes, and support data-driven decision-making.

---

## Project Objectives

- Analyze hospital readmission trends
- Identify factors influencing patient readmissions
- Perform data cleaning and preprocessing
- Query healthcare data using SQL
- Conduct exploratory data analysis using Python
- Build interactive Tableau dashboards
- Generate business insights for healthcare stakeholders

## Business Insights

- Readmission rates vary across different admission types.
- Longer hospital stays are associated with higher readmission risk.
- Patient demographics and clinical procedures influence readmission patterns.
- Interactive dashboards enable stakeholders to monitor hospital performance effectively.
