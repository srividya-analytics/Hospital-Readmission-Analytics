# Python Analysis

This folder contains Python-based analytical outputs generated during the Hospital Readmission Analytics project.

## Python Outputs Included

- Data Mining for Business Decisions (HTML report)
- Business Forecasting & Visualization (HTML report)
- Age-wise Early Readmission Rate (CSV)
- Forecasting Summary Table (CSV)

Python was used for:

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Business forecasting
- Exporting analytical reports

These outputs complement the SQL queries and Tableau dashboards included in this repository.
